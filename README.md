# Passenger Counter App

A simple web application that counts how many people enter a station.  
This project was built using **HTML, CSS, and JavaScript** to practice basic DOM manipulation and event handling.

## Features

- Increment the passenger count
- Save previous counts
- Reset the counter after saving
- Simple interactive button animations
- Background image styling

## How It Works

1. Click **INCREMENT** to increase the passenger count.
2. Click **SAVE** to store the current count.
3. The saved number will appear in **Previous entries**.
4. The counter resets back to **0** after saving.
